# lvgl_cmake_pro

#### 介绍
LVGL是一个轻量级通用型图形库，适合在嵌入式设备上使用，但是日常过程中，我们在调试嵌入式设备的时候会有很多硬件bug，并且每次烧写代码非常麻烦，所以一个好的仿真软件非常重要，笔者在别人的基础上移植到win10系统，测试使用正常，初次记得删掉build目录运行
